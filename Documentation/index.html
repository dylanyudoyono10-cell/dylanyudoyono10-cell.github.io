1. Introduction

Thank you for purchasing this exclusive portfolio template! This website is built using standard modern web technologies: HTML5, CSS3, and Vanilla JavaScript. It also comes equipped with third-party libraries for smooth icons and animations.

The template is fully responsive (mobile-friendly) and completely ready to showcase your skills, tools, services, and portfolio projects in a highly professional manner.

3. Content Customization

Open the Code/index.html file using your preferred text editor (such as VS Code, Notepad++, or Sublime Text) to edit the basic text layout.

A. Changing the Main Title / Name

Locate the following block of code within the <div class="hero"> section and edit the text:

<h1>
Hello, i'm 
<br><span>Your Name Here</span>
</h1>

B. Editing the About Me Section

Find the <div id="about" class="about"> section to modify your short biography statement inside the <p> tags.

C. Updating Social Media Links

In the Footer section at the bottom of the page, replace the # symbol inside the <a href="#"> tags with your actual social media URLs:

<div class="social">
    <a href="https://instagram.com/yourusername"><i class="fa-brands fa-instagram"></i></a>
    <a href="https://linkedin.com/in/yourusername"><i class="fa-brands fa-linkedin"></i></a>
</div>

4. Replacing Images

To completely personalize this website, you must update the image assets stored inside the Picture/ directory.

Image Component Location & Default Filename Recommended Format & Dimensions
Hero Profile Photo Picture/personal_photo.png Transparent PNG (No Background), circular or square orientation.
Portfolio Project Images Picture/portfolio_image/portfolio.jpg JPG/PNG, 16:9 Landscape Aspect Ratio (Recommended resolution: 800x450 pixels).
Skill Icons (Tools) Picture/Tools_Image/ Transparent PNG or square format with the logo perfectly centered.
💡 IMPORTANT TIP: The easiest way to swap images is to overwrite the old files with your new ones while keeping the exact same filename and file extension (e.g., replace the profile picture and name it personal_photo.png). If you use a different name or extension, remember to update the corresponding <img src="..."> tags inside your index.html file.

5. Features & Animations

This website uses external plugins to create a dynamic user experience:

  • FontAwesome Icons: Handles the mobile hamburger menu, back-to-top arrow, and social icons. Loaded via a CDN script in the HTML header.
  • AOS (Animate On Scroll): Generates smooth fade-in animations as you scroll down the page. Controlled directly using attributes like data-aos="fade-up".
  • Scroll to Top: An orange circular button that automatically appears in the bottom right corner once a user scrolls past 550px, offering an easy way to return to the header.

6. Frequently Asked Questions (FAQ)

Q: Why are my images or logo icons not showing up?

A: Make sure your HTML file resides within the Code/ folder and the Picture/ folder is right outside next to it. The code loads images relatively via ../Picture/ (going back one directory first).

Q: How do I add a new project card to the Portfolio section?

A: Simply copy one of the existing <div class="box"> elements inside the main <div class="portfolio-box"> container, then update the internal text and image source.

Q: Can I change the main accent color?

A: Absolutely! Open up style.css, find the :root selector at the top, and edit the hex color codes assigned to variables like --primary2, --hover, or --active to match your brand identity.
